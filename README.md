# The History of AI

An interactive timeline of the **50 most influential research papers** in the history of artificial intelligence, machine learning, and computer science — from the first mathematical model of a neuron (1943) to the foundation models of today (2023).

Each paper is a node on a 3D, depth-aware timeline. Click any card to expand its summary, venue, and a link to the original paper.


## Editing the content

All paper data lives in `papers.js` as an array of objects. To add, remove, or edit an entry, change the array — the timeline and node numbers (01–50) are generated automatically.

```js
{
  year: 2017,
  title: "Attention Is All You Need",
  authors: "Ashish Vaswani et al. (Google)",
  venue: "NeurIPS",
  tag: "Transformers",
  blurb: "Introduced the Transformer, replacing recurrence entirely with self-attention…",
  url: "https://arxiv.org/abs/1706.03762",
}
```

## Notes

The selection is a curated list of foundational work rather than a strict citation-count ranking. Suggestions and swaps are welcome.
